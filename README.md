# Первая часть тех. задания

## Установка зависимостей
```pip3 install poetry```

```poetry install```



## Запуск проекта
```pytest main.py```

Запуск тестов
```pytest``` или ```pytest -m {mark}```

Доступны марки: 
```    
    engine
    engine_init
    engine_color
    engine_shape
    engine_draw
    engine_clear
    shape
   ```
